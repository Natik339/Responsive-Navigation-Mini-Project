# Responsive Sidebar Menu using HTML & CSS

A simple and responsive sidebar navigation mini project built using only **HTML** and **CSS**.
This project demonstrates the use of positioning, transitions, hover effects, icons, and the checkbox hack technique without using JavaScript or any backend.

---

## 📌 Features

* Pure HTML & CSS
* Responsive sidebar navigation menu
* Smooth open/close sidebar animation
* Hover effects and transitions
* Font Awesome icons integration
* Google Fonts used
* No JavaScript used
* Beginner-friendly frontend mini project

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
* Google Fonts
